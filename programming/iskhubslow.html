Introducing the @IsKHubSlow bot!

Posted last March 9, 2021

There's a new project in town! Introducing my new bot, @IsKHubSlow!

About the Project

IsKHubSlow is a bot that checks if KHub is slow based on its response time and tweets it, inspired by the idea of @IsKHubDown on Twitter. Open source on GitHub! (Not affiliated to KHub or PSHS.)

Features

  • Requests data from an API that checks for response times
  • Determines if current response time is slow using historical data and statistics
  • Condense information and tweets the results

Built With

  • Node.js
  • Twitter API v2
  • UptimeRobot API
  • Replit database

Visit the official site at xyr.codes/khubslow for more or start contributing on Github at xyr11/iskhubslow!

Follow @IsKHubSlow on Twitter to see what he's been tweeting!
